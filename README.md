# Decentralized Public School Bus Transportation System

A comprehensive blockchain-based solution for managing school bus transportation using Clarity smart contracts on the Stacks blockchain.

## System Overview

This system consists of five interconnected smart contracts that manage different aspects of school bus transportation:

1. **Route Optimizer** - Plans efficient bus routes based on student addresses and school locations
2. **Student Safety Tracker** - Monitors student boarding and departure at designated stops
3. **Driver Verification** - Validates bus driver licenses, training, and background checks
4. **Vehicle Maintenance** - Tracks bus maintenance needs and ensures safety compliance
5. **Parent Notifications** - Provides real-time updates on bus delays and route changes

## Key Features

- Decentralized route optimization and management
- Real-time student safety tracking
- Automated driver qualification verification
- Predictive vehicle maintenance scheduling
- Instant parent notification system
- Immutable audit trails for all activities
- Emergency response capabilities

## Installation

1. Install Clarinet
2. Clone this repository
3. Run clarinet check to validate contracts
4. Run npm test to execute test suite

## Usage

Each contract can be deployed independently and provides specific functionality for the transportation system.
